Air Tracking

Visualização e análise de viagens aéreas

Com base na API OpenSky
Visitar repositório do projeto

Code UA Project Page

Controlo aéreo

Sistema composto por um conjunto de dashboards com base em dados fornecidos pela API OpenSky sobre aviões e respetivas viagens

 

Features

  • Visualização de viagens aéreas

    A plataforma AirTacking permite a visualização de voos em tempo real

  • Filtragem de viagéns aéreas

    Possibilidade de filtragem do número de viagens aéras numa determinada região

  • Análise de viagens aéreas

    Informações detalhadas sobre voos e visualização da trajetória do voo

  • Alertas sobre viagens aéreas

    A plataforma notifica os utilizadores sobre o estado de determinados voos

Features

 

Funcionalidade

Obter uma plataforma interativa de visualização de dados analíticos acerca de voos a ocorrer em tempo real. Poderão obter mais dados estatísticos acerca de voos, aviões e aeroportos bem como receber alertas de descolamentos de voos

Cenário:

Obter informação acerca de um voo

Dado que o utilizador esteja na página de visualização de voos,

Quando o utilizador clica sobre o ícone indicativo de um voo a decorrer

Então o utilizador deverá ler as informações de velocidade, país de origem, entre outras informações

Cenário:

Obter dados estatísticos de um voo em específico numa janela temporal específica

Dado que o utilizador esteja na página de estatísticas

E que o utilizador preencha os campos "2019-03-20" e "2019-04-10" que definem a janela temporal de filtro de pesquisa

E que o utilizador preencha o campo relativo ao voo com icao24 "29bc123"

Quando o utilizador clica sobre o botão de pesquisa

Então deverá obret as informações estatísticas acerca do voo "29bc123" entre as datas de "2019-03-20" e "2019-04-10"

Cenário:

Receber um alerta de descolamento de um voo definido

Dado que um alerta está definido para o descolar do voo "29bc123" pelo utilizador de email albertomartins@gmail.com

Quando o voo "29bc123" descolar

Então o utilizador de email arbertomartins@gmail.com deverá receber uma notificação da partida do voo

Cenário:

Filtrar no mapa os voos por país de origem

Dado que o utilizador esteja na página de visualização de voos,

E que o utilizador clique na secção de filtros de vizualização

E que o utilizador defina o dropdown com o nome "País" para "Portugal"

Quando o utilizador clicar no botão "Filtrar"

Então o utilziador deverá deixar de ver todos os voos cujo país de origem não é Portugal

Cenário:

Obter informação acerca de um aeroporto

Dado que o utilizador esteja na pagina de visualização de voos,

Quando o utilizador clicar sobre o ícone indicativo de um aeroporto no mapa

Então o utilizador deverá obter informações sobre esse aeroporto
